# iot-bank-locker-android-otp
IoT Based Advanced Bank Locker System for Enhanced Security using Android OTP Authentication
🔐 IoT Based Advanced Bank Locker System for Enhanced Security Using Android-Based OTP System

Overview

This project presents an IoT-Based Advanced Bank Locker System designed to improve locker security through Android-based OTP authentication. The system provides secure access control by generating a One-Time Password (OTP) and verifying it before unlocking the locker.

The project uses IoT technology for monitoring and communication, enabling real-time notifications and improved protection against unauthorized access.

---

Problem Statement

Traditional bank locker systems mainly depend on physical keys and passwords, which may lead to security risks such as key duplication, password theft, and unauthorized access.

This project introduces an OTP-based authentication mechanism integrated with IoT technology to provide a more secure and reliable locker system.

Working of the System

Step 1 – User Access Request

User requests access through Android application.

Step 2 – OTP Generation

System generates OTP and sends it to registered user.

Step 3 – OTP Verification

User enters OTP.

Step 4 – Access Decision

- Correct OTP → Locker Opens
- Wrong OTP → Access Denied

Step 5 – Notification

System sends status notification and records activity.
